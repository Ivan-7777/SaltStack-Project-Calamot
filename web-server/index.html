Hola, bienvenidos al hosting web

Para empezar en nuestros primeros pasos podemos usar los scripts en Scripts/ para crear nuestro primer host web.

Podemos probarlo editando el archivo hosts de nuestra máquina real poniendo nuestra ip_mv_web y nuestro nombre de host web creado con el script.

Luego dirígete al navegador y busca https://host_web_name.com o conéctate vía SFTP usando las credenciales creadas en el script.
